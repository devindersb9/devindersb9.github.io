HTML tagsDescriptionUsage
⁢ < title> Defines a title for the documentA title tag is an HTML element that specifies the title of a web page.
⁢< body>Defines a list itemThe body tag element contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc.
⁢< li>Roland MendelThe tag is used in order to list unordered lists, nd menu lists

⁢< b>Defines bold textThe "b" tag is used as the last resort when no other tag is appropriate
⁢ < area>Defines an area inside an image-mapThe area tag defines an area inside an image-map
⁢ < div>Defines a section in a documentThe div tag is uesd as a container for the other HTML to style them with CSS or to persorm certain task with JavaCcript
⁢ < hr>Defines a thematic change in the contentThe hr element is used to seperate content in HTML page
⁢ < iframe>Defines an inline frameiframed tag is used to embed another document within the current HTML page

⁢ < dl>Defines a description listThe dl tag is used in conjuction other tags
⁢ < dfn>Represents the defining instance of a termThe defining instance is often the first use of a term in a document.
⁢ < dt>Defines a term/name in a description listThe dt tag is used in conjunction with defining description list and describes/term each name
⁢ < figure>Specifies self-contained contentThe figure tag element is related to the main flow, its position is independent of the main flow, and if removed it should not affect the flow of the document.
⁢ < br>Defines a single line breakThe br tag is an empty tag which means that it has no end tag

⁢ < dd> Defines a description/value of a term in a description list The dd tag is used to describe a term/name in a description list.
⁢ < map> Defines a client-side image-map The image tag is used to define a client-side image-map. An image-map is an image with clickable areas.
⁢ < ol> Defines an ordered list The ol tag defines an ordered list. An ordered list can be numerical or alphabetical.
⁢ < u> Defines text that should be stylistically different from normal text The u tag represents some text that should be stylistically different from normal text, such as misspelled words or proper nouns in Chinese.
